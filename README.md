

# 🚀 PingUp – Social Media App (MERN Stack)

A full-stack social media application built using **MongoDB, Express, React, and Node.js**, featuring real-time chat, posts, stories, and media uploads.


## ✨ Features

* 🔐 Secure User Authentication (Sign-up, Sign-in, Profile Management)
* 💬 Real-time Chat & Messaging with Notifications
* 📝 Post Feed with Text & Image Content
* 📸 Story / Status Uploads
* 👥 Follow / Unfollow & Friend Requests
* 🗄️ Media Storage & Optimization using **ImageKit**
* ⚡ Background Jobs handled via **Inngest**
* 🔎 User Search and Discovery


## 🛠️ Tech Stack

* **Frontend:** React.js
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Authentication & Media:** Clerk, ImageKit
* **Background Jobs:** Inngest
